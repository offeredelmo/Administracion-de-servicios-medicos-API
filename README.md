# Salud API
1. Clonar el proyecto
2. ``` npm install ```
3. clobar el archivo ```.env.template ``` y renombrarlo a .env
4. cambiar las variables de entorno
5. Levantar base de datos para dev

``` 
docker-compose -d up
```
